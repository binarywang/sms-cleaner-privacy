智能短信助手隐私政策

更新日期：2026 年 6 月 23 日

生效日期：2026 年 6 月 23 日

本隐私政策适用于“智能短信助手”（以下简称“本应用”）。本应用是一个短信整理工具，用于在用户授权后读取本机短信，帮助用户通过 AI 分类识别可能适合清理的短信，并由用户自行审核和确认是否删除。

请在使用本应用前仔细阅读本政策。若您不同意本政策，请停止使用本应用。

  • 应用名称：智能短信助手
  • 包名：com.binarywang.smscleaner
  • 开发者/运营者：Binary Wang
  • 联系邮箱：a@binarywang.com

二、本应用如何处理个人信息

1. 短信信息

在您授予短信权限后，本应用会读取本机系统短信，用于展示短信会话、统计短信数量、进行 AI 分类分析、标记可能适合清理的短信，以及在您确认后删除选中的系统短信。

可能处理的信息包括：短信正文、发件人/收件人号码、短信时间、短信状态、短信 ID、会话信息、SIM 卡信息、AI 分类结果、删除确认结果。

本应用不会自动删除短信。AI 只提供分类和清理建议，最终删除操作必须由用户主动勾选并二次确认。

2. 通讯录信息

在您授予通讯录权限后，本应用会读取通讯录联系人名称和号码，用于在短信列表中显示联系人姓名。若您不授予通讯录权限，本应用仍可使用短信号码展示会话。

本应用不会上传通讯录信息。

3. AI 分析相关信息

当您主动点击 AI 分析功能时，本应用会将待分析短信内容发送到您在设置中配置的 AI 接口地址，用于生成短信分类、原因和是否建议清理的结果。

默认接口地址为 https://api.minimax.chat/v1，您可以在应用设置中修改 Base URL、API Key 和模型名称。请确认您信任所配置的 AI 服务提供方，并阅读该服务提供方的隐私政策和服务条款。

本应用不会将短信内容上传到开发者自有服务器。短信内容仅会在您主动发起 AI 分析时发送到您配置的 AI 接口。

4. API Key 和模型配置

您在设置中填写的 API Key、Base URL、模型名称等信息会保存在本应用私有存储中，用于后续发起 AI 分析请求。

本应用不会将 API Key 上传到开发者自有服务器，也不会在日志中记录 API Key。

5. 本地分析记录和删除记录

本应用会在私有存储中保存已分析短信 ID、已分析短信快照、AI 分析日志元数据和最近删除短信的本地记录。

其中短信快照和删除记录可能包含短信正文，属于敏感个人信息，仅保存在本机应用私有存储中。AI 分析日志只记录分析元数据，不记录短信正文、API Key 或完整 AI 请求/响应载荷。

三、权限使用说明

  • 读取短信：用于读取和展示本机历史短信，并进行 AI 分类分析。
  • 写入/接收/发送短信：用于满足 Android 默认短信应用相关能力要求，支持短信删除、接收占位和基础短信发送能力。
  • 读取通讯录：用于将短信号码匹配为联系人姓名。
  • 网络访问：用于连接您配置的 AI 接口完成短信分类分析。
  • 桌面角标权限：用于在支持的华为桌面显示未读短信数量角标。

您可以在系统设置中关闭相关权限。关闭权限后，部分功能可能无法正常使用。

四、信息存储和保护

本应用将数据保存在应用私有存储中。除系统备份机制外，其他应用通常无法直接访问这些数据。

本应用不会主动将短信正文、通讯录、API Key 或删除记录上传到开发者服务器。

由于 AI 分析需要调用您配置的外部 AI 服务，请您谨慎选择可信服务商，并妥善保管 API Key。

五、信息共享、转让和公开披露

本应用不会向开发者自有服务器收集、出售、出租或公开披露您的短信、通讯录、API Key 或本地分析记录。

当您主动使用 AI 分析功能时，待分析短信内容会发送至您配置的 AI 服务提供方。该行为是实现 AI 分类功能所必需的。

除法律法规要求、司法机关或行政机关依法要求外，本应用不会向其他第三方共享您的个人信息。

六、用户控制和删除

  • 在系统设置中撤销短信、通讯录、网络等权限；
  • 在应用内删除已保存的本地删除记录；
  • 清除应用数据或卸载应用，以删除本应用私有存储中的配置、分析记录和本地删除记录；
  • 在 AI 服务提供方后台管理或删除与 API Key 相关的远端数据，具体以该服务提供方规则为准。

七、未成年人保护

本应用不面向未成年人提供专门服务。未成年人使用本应用前，应取得监护人同意，并在监护人指导下使用。

本政策可能因功能调整、法律法规或应用市场要求而更新。更新后会在应用介绍页或公开页面中展示新的隐私政策。
